# 🔘 pure switch — однокнопочное управление роборукой

Минималистичный веб-рубильник для удалённого включения/выключения робота или отдельного модуля (например, питания схвата, контроллера, реле).  
Интерфейс состоит из одной кнопки и светодиода, серверная часть на Flask сохраняет состояние в JSON‑файл.  
Никакой телеметрии, слайдеров, сложной логики — только **вкл / выкл**.

---

## 📦 Что внутри

| Компонент | Назначение |
|-----------|------------|
| `index.html` | Фронтенд: кнопка ON/OFF, индикатор, запросы к серверу |
| `server.py` | Бэкенд на Flask: приём команд, запись/чтение `power_state.json` |
| `power_state.json` | Файл состояния (создаётся автоматически при первом переключении) |

---

## ⚙️ Как это работает

1. Браузер загружает страницу и запрашивает текущее состояние (`GET /power`).
2. Сервер читает `power_state.json` (если файла нет, считается что выключено).
3. Пользователь нажимает кнопку — JavaScript отправляет `POST /power` с телом `{"state": true/false}`.
4. Сервер записывает новое состояние в `power_state.json` и возвращает подтверждение.
5. Интерфейс обновляется (текст кнопки, цвет, светодиод).

При перезапуске сервера или перезагрузке страницы состояние сохраняется, потому что оно лежит в файле.
